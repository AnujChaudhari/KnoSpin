# Quick Shop - Mobile E-Commerce

A complete, modern e-commerce website built to run perfectly on mobile devices. Entirely free to build and host.

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS, Framer Motion
- Firebase (Auth, Firestore, Storage)
- Cloudinary (free image hosting)
- Razorpay Test Mode
- EmailJS (order emails)
- Hosted on Vercel

## Setup
1. Clone the repo.
2. Copy `.env.local.example` to `.env.local` and fill all keys.
3. Install: `npm install`
4. Run: `npm run dev`

## Deploy on Vercel
1. Push to GitHub.
2. Import on Vercel, set environment variables.
3. Deploy.

## Admin Panel
- Access at `/admin` after adding your UID to Firestore `admins` collection.
